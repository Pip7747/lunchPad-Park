## 🎵 What Is lunchPad-Park?

**lunchPad-Park** is a fun, interactive, web-based launchpad that lets you create beats and rhythms right in your browser — no downloads, no DAW, no gear required. Just open it up, map your keys, and start jamming.

Whether you're a bedroom producer, a curious beginner, or just someone who likes pressing buttons to make satisfying sounds — this one's for you. 🥁

---

## ✨ Features

- 🎹 **Keyboard-driven** — trigger sounds with customizable key bindings
- 🎛️ **Multiple pad layouts** — configure your launchpad the way you like it
- ⚡ **Instant response** — low-latency sound playback in the browser
- 🌐 **Zero install** — runs entirely in the browser, no dependencies needed
- 🎨 **Visual feedback** — pads light up when triggered so you can follow the rhythm

## 🚀 Getting Started

### Play It Now

Just open `index.html` in your browser and start pressing keys!

### Clone & Run Locally

```bash
git clone https://github.com/Pip7747/lunchPad-Park.git
cd lunchPad-Park
# Open index.html in your browser
open index.html   # macOS
xdg-open index.html  # Linux
start index.html  # Windows
```

No build step. No `npm install`. Pure web magic. ✨

## 🎮 How to Use

| Action | How |
|---|---|
| Trigger a pad | Press the assigned keyboard key |
| Change key mapping | Use the settings panel to remap keys |
| Switch pad layout | Select a different layout from the menu |
| Reset to defaults | Refresh the page |

> **Tip:** Try mapping pads to the home row keys (`A S D F J K L ;`) for the most natural drumming feel!
